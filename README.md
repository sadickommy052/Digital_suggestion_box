# Digital Suggestion Box

## Project Description
Digital Suggestion Box is a web-based system that allows users to submit, manage, and review suggestions online. It helps organizations collect feedback efficiently and manage user suggestions in a structured way.

## Features
- User registration and login system
- Submit suggestions
- Admin dashboard for managing users and suggestions
- Role-based access (Admin, Manager, User)
- Secure database integration
- Easy to use interface

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Project Structure
- admin/ → Admin dashboard and management files
- dashboard/ → User dashboard features
- config/ → Database configuration
- login.php → Login system
- register.php → User registration
- index.php → Main landing page
